Academic Project

IOT Based Biometric Attendance System

This project aims at designing a smart attendance system that monitors and manages attendance of the students in an institution efficiently. We interfacing fingerprint sensor with Arduino UNO Microcontroller, LCD display and RTC Module. We used fingerprint Module and Arduino to take and keep attendance data and records. For every user ID will be displayed according to your fingerprint input.

Arduino is an open source electronics platform based on easy to use hardware and software. RTC Module is used to leeps track of seconds, minutes, hours, day,date, month and year data. So, we designed IOT Based Biometric Attendance Syste, using Arduino.

The Internet of Things describes the devices with sensors, processing ability, software and other technologies that connect and exchange data with other devices and systems over the internet or other communication networks

How its Work :

To enroll New fingerprint Click on the Enroll button. then select the memory location where you want to store yout fingerprint using the UP/DOWN button.
then click on Ok. Put your finger and remove your finger as the LCD instructs.Put your finger again. So finaaly your fingerprint gets stored.

Conclusion :
The Fingerprint Based Biometric Attendance System using Arduino represents a significant leap in attendance tracking technology.

By leveraging the uniqueness of fingerprint, this system provides an accurate, efficient, and secure solution for managing workforce attendance.
